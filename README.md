Парикмахерская оказывает услуги стрижки, окрашивания, укладки и другие парикмахерские процедуры. Пользователями системы являются администратор (управляет справочниками, сотрудниками и расписанием) и мастера (просматривают свои записи, отмечают выполнение). Клиенты обслуживаются через администратора, который регистрирует их и создаёт записи.

Пользователи системы делятся на 2 категории:

 Администратор — управляет учётными записями мастеров, списком услуг, создаёт и редактирует записи клиентов, просматривает статистику.
 Мастер — видит своё расписание на текущий день, может отметить запись как выполненную или отменённую, просматривает детали клиента и выбранные услуги.

 1.2 Обоснование выбора предметной области

Предметная область выбрана по следующим причинам:

 Наличие логики, которая естественно ложится на триггеры и процедуры: проверка пересечений времени при записи, автоматический пересчёт общей стоимости, логирование изменений статусов.
 Количество таблиц (6) обеспечивает разнообразие связей: 1:N (роли→пользователи, клиенты→записи) и M:N (записи↔услуги через связующую таблицу).
 Практическая актуальность: любая парикмахерская нуждается в удобном инструменте для управления расписанием и учёта оказанных услуг.
 Предметная область может служить основой для дипломного проекта — её легко расширить онлайнзаписью, программами лояльности, учётом расходных материалов.
